# Prompt Universe Router

Canonical router: `.agents/workflows/prompt-universe.md`

Use this root file only as a discoverable entrypoint for Codex autocomplete or
manual lookup. Do not duplicate the full router here.

When a user prompt is vague, raw, multi-intent, or does not name the right
skill, read `.agents/workflows/prompt-universe.md` before choosing a workflow.

Source of truth:

- Agent entrypoint: `AGENTS.md`
- Agent operating layer: `.agents/`
- Prompt router: `.agents/workflows/prompt-universe.md`
